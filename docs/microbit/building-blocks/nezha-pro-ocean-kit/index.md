# 哪吒pro海洋套件

## 简介
哪吒 Pro 海洋主题套装以编程积木为载体，以小恩海洋冒险故事为主线，帮助学生探索海洋生态与生物多样性。课程涵盖海洋生物生存技能，以及潮汐发电、海洋养殖等人类对海洋的开发利用实例。学生将深入探究海洋资源开发面临的挑战，了解人类活动对海洋生态的影响，并对这些问题尝试提出解决方案。套件旨在提升学生海洋认知，激发探索兴趣，培养海洋保护意识。
