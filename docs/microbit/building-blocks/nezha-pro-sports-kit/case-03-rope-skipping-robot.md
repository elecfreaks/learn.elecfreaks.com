## 扩展知识


**传动比与速度关系**

传动比公式：主动轮齿数 : 从动轮齿数。例如，24齿齿轮带动8齿齿轮时，从动轮转速为主动轮的3倍。

**动力与扭矩转换**

小齿轮驱动大齿轮时，速度降低但扭矩增加（适用于需要更大力量的场景）

大齿轮驱动小齿轮时，速度增加但扭矩降低（适用于需要更快速度的场景）
