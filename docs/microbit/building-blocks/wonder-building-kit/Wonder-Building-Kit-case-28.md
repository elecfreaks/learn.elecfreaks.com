![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/building-blocks/wonder-building-kit/images/Wonder-Building-Kit-case-21-02.png)

在搜索框中搜索wukong，点击图片中wukong，添加扩展包。
