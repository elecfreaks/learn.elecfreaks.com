## 结果

开启之后，
当按下micro:bit上面的按键A时，叉车向前行驶；
当按下micro:bit上面的按键B时，叉车向后行驶；
当按下micro:bit上面的按键A+B时，叉车停止行驶；
当按下micro:bit上面的徽标时，抬升车叉；
当松开micro:bit上面的徽标时，降低车叉。
