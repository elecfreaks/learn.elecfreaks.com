# Cutebot Pro积木包


# 产品简介

为了满足 Cutebot Pro 用户对智能互动体验的追求，我们制作了它的专属积木扩展包。使用该产品可以让cutebot pro实现与简易机械爪和AI摄像头相结合的案例。

**注意：此扩展块包专为 Cutebot Pro 设计，仅包含用于连接简易机械爪或 AI 摄像头的结构连接部件。並不包含简易机械爪以及AI摄像头（需要额外购买）。**
