# Jacdac Series

# 简介
jacdac（全称 Just Another Connected Device Access Controller）是微软研究院开发的跨平台硬件通信协议与开发框架，旨在通过标准化接口和工具链，简化教育机器人、物联网设备及物理计算项目的开发流程。其核心设计理念是即插即用、低代码编程与多设备协同，尤其适合教育场景中的快速原型开发与跨学科实践。
