# 抛硬币

## 简介

在足球比赛中，通常由裁判员通过抛硬币的方式来决定哪个队伍首先发球。在比赛开始前，裁判员会召集两队的队长，然后进行抛硬币，赢得抛硬币的队伍可以选择是先开球还是选择场地。如果他们选择先开球，那么对方队伍就会选择场地；反之亦然。这种抛硬币的方法是公平的，可以确保比赛的公正性。

## 教学目标

了解足球比赛的基本规则。

通过micro:bit来制作抛硬币的案例。

培养问题解决能力。
