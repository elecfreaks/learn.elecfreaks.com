# 正式比赛

## 简介

标准的足球比赛每队有11名球员上场，包括1名守门员和10名场上球员。在正式的国际比赛和大多数职业联赛中，这个人数是固定的。
但由于场地和小车数量的考虑，我们这次项目中只考虑1V1的小车足球对抗赛。

## 教学目标

了解足球比赛的基本规则。

使用趣味足球赛套装制作比赛小车。

培养问题解决能力。
