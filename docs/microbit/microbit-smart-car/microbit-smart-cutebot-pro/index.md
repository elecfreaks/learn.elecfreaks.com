# Cutebot Pro智能赛车简介

## 简介
---

Cutebot Pro是一个用于STEAM教育的编程机器人，它配备了四路红外巡线传感器、编码电机、LED彩虹灯和超声波传感器等多种设备。通过编码电机，Cutebot Pro能够灵活地控制小车前进的距离并实现更精确的转弯角度控制。同时，四路巡线传感器使其能够在巡线的过程中识别十字路口和其他各种更复杂的路线。Cutebot Pro还提供了多种扩展接口，可以扩展更多硬件，充分激发孩子的想象力与创造力。
