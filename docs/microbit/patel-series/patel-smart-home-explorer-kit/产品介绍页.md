# 产品介绍

## 简介

micro:bit 智能家庭探索者套装
专为8-16岁青少年设计的动手实践套件，通过传感器、舵机与可拼装结构件，搭建智能灯光、自动晾衣架、温控风扇等生活化装置。从硬件连接到逻辑控制，学习基础编程与电子工程原理，在真实场景中培养创造力与解决问题的能力，让科技探索回归生活本质。

**用micro:bit搭建你的智能家居装置，在动手创造中解锁科技奥秘！**
