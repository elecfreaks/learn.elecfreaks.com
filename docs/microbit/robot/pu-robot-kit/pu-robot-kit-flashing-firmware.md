# 烧录固件

如果你需要更新PU机器人的固件请按以下步骤执行：

1. 获取固件

PU Robot固件下载：
遥控手柄固件下载：

2. 将固件分别下载到PU机器人和遥控手柄对应的两个micro:bit.

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

  **注意：**注意区分两个micro:bit，防止固件错误导致PU机器人无法启动

3. micro:bit拆卸与安装

    1. 将PU机器人头部旋转90°

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    2. 拧下固定螺丝

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    3. 图三：打开头盖

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    4. 图四：安装micro:bit

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    5. 图五：装好头盖

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    6. 图六：将固定螺丝拧回

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)


  **警告:**PU机器人通电时,请勿强行扭动其关节，这将对伺服电机造成损坏。

2. 将micro:bit安装至遥控手柄

    1. 图一：拧下手柄背面亚克力固定螺丝

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    2. 图二：安装micro:bit

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    3. 图三：将固定螺丝拧回

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)
