# 快速上手

## 开箱指南

1. 从包装材料中取出机器人 PU 和游戏手柄（如果你选装了micro:bit，则micro:bit已经安装到机器人 PU 和游戏手柄中）
2. 打开外包装盒，取出产品使用手册
3. 在机器人背包（背部储物仓）中找到配件包，内含 2 节 AAA 电池
4. 找到螺丝刀（用于电池安装）
5. 确认所有物品完好无损

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-quick-start-01.png)

## 安装电池

1. 找到游戏手柄背面的电池盖
2. 使用螺丝刀拧开电池盖螺丝
3. 装入2节AAA电池（注意正负极方向）
4. 盖回电池盖并拧紧螺丝

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-quick-start-02.png)

**安全提醒：**
- 请使用全新碱性电池
- 勿混用新旧电池、不同类型电池
- 长时间不使用请取出电池

## 遥控手柄开机操作

1. 找到手柄侧面的电源开关
2. 将开关拨至 "ON" 档位
3. micro:bit LED 屏显示当前无线电控制频道号

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-quick-start-03.png)

## 关机操作
1. 将电源开关拨至 "OFF" 档位
2. LED 显示屏熄灭

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-quick-start-04.png)

**省电建议：**长时间不使用时，请关闭手柄电源以节省电量

## PU Robot开关机操作
1. 找到机器人下巴处的黑色电源键
2. 开机：轻按电源键（短按一下）/关机：轻按电源键（短按两下）
3. 开机后等待2秒校准完成
**位置示意：**电源键位于机器人正面下巴位置

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-quick-start-05.png)

## 充电操作

1. 找到机器人下巴处的USB充电接口
2. 插入USB数据线开始充电

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-quick-start-06.png)

## 操作指南

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-quick-start-07.png)


| 序号 | 控制项 | 功能说明 |
| :--- | :---- | :--- |
| 1    | 操纵杆                  | 1. 向前/向后推动操纵杆，机器人将向对应方向移动，推动幅度越大，机器人移动速度越快<br />2. 向左/向右倾斜操纵杆，机器人将向对应方向转向，倾斜幅度越大，转向角度越大<br />3. 按下操纵杆一次，机器人进入休息模式<br />4. 将操纵杆大幅度向左/向右倾斜，机器人将向对应方向横移 |
| 2    | Micro:Bit 手势控制      | 使用操纵杆控制时，向上/向下倾斜手柄，可控制Pu的头部上下摆动；向左/向右倾斜手柄，可控制Pu的头部向对应方向转动   |
| 3    | B1 按钮                 | 让Pu机器人进入自动驾驶模式，可自主行走或探索迷宫         |
| 4    | B2 按钮                 | 让Pu机器人跳跃，适用于足球游戏和舞蹈场景。请勿连续按压此按钮，需等待上一次跳跃动作完成后，才能执行下一次跳跃   |
| 5    | B3 按钮                 | 让Pu机器人跳舞。开启音乐后，Pu会跟随音乐节奏跳舞，其LED灯也会随节拍变换颜色               |
| 6    | B4 按钮                 | 让Pu机器人踢球，适用于足球游戏场景。请勿连续按压此按钮，需等待上一次踢球动作完成后，才能执行下一次踢球       |
| 7    | Micro:Bit 感应Logo     | 触摸Micro:Bit的Logo区域，Pu机器人会随机说话或唱歌，内容取决于它近期的活动状态，可通过编程修改出厂默认行为  |
| 8    | Micro:Bit A键           | 增加无线电控制频道，频道编号会同步显示在Micro:Bit的屏幕上           |
| 9    | Micro:Bit B键           | 减少无线电控制频道，频道编号会同步显示在Micro:Bit的屏幕上           |
| 10   | Micro:Bit 复位键        | 重启手柄控制程序，无线电控制频道将恢复为默认值6        |
| 11   | Micro:Bit A+B组合键     | 同时按下A键和B键，显示当前实际无线电控制频道（实际频道号=屏幕显示数值+160）           |
| 12   | 电源开关                | 开启/关闭手柄电源                                    |
