# 太阳能电源管理模块

## 简介

这款太阳能电源管理模块支持太阳能板和Type-C充电，具备电源输出控制开关，内置电池仓，可安装18650电池扩展蓄电能力。


![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/sensor/octopus-sensors/input/images/03461-1.png)


## 特性

支持太阳能/Type-C进行充电
内置电池仓
具备电源输出控制开关
