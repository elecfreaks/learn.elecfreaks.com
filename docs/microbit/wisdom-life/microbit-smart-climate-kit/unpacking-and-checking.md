# 开箱自检

进行开箱自检是为了确保您购买的产品符合您的预期并且没有受到任何损坏。请您根据以下步骤来进行开箱自检操作。

## 1、检查包装

检查包装是否完好无损，确认包装上的标签和描述与您购买的产品相符。

## 2、确认配件

确认产品附带的所有配件是否全部齐全，并与产品清单中的描述相符。
