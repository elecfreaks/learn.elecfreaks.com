# Lesson 36 Wireless Controlled Alarm Door
---
- Enhance the security of your door with the micro:bit. This tutorial was written by Sean Lew from the Singapore University of Technology and Design during his summer internship in 2018.

## Objectives
---
- 1. Build your own wireless alarm door!
- 2. Set up an alarm for any intruders or unlocked doors!

## Materials
- 2 x Micro:bit
- 1 x Breakout Board
- 1 x Collision Sensor
- 1 x Buzzer

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/getting-started/microbit-tinker-kit/images/Boaq8eL.jpg)

### Step 1 - Setup!
- Insert one micro:bit into the breakout board.
- Connect the buzzer to pin 0 of the breakout board, and connect the collision sensor to pin 1.
- Make sure to download the "tinkercademy-tinker-kit" extension before proceeding to Step 2.

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/getting-started/microbit-tinker-kit/images/VvnoXBN.jpg)

### Step 2 - Signal Transmitter
- Drag these code blocks into the MakeCode platform, then download them to the micro:bit connected to the breakout board.
- Press the collision sensor a few times to check if your code is working properly!

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/getting-started/microbit-tinker-kit/images/QawJArj.png)

### Step 3 - Signal Receiver
- Download this set of code for the second micro:bit (receiver).
- Now test the collision sensor to see if the receiving micro:bit is working properly!
- Now that you have set up your first wireless door security device, you can try it with more than one door!
- Feel free to add other forms of alarms to the receiving micro:bit to enhance your own security.
