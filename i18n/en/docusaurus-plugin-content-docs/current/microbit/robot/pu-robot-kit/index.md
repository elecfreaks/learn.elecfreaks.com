# micro:bit PU Robot Kit(EF08449)

## 简介
---

PU 机器人是专为 STEAM 教育打造的智能教育机器人，采用模块化结构设计，兼容积木搭建。支持 GPIO、IIC 等标准接口与传感器扩展，配备 LED 大灯、超声波传感器。出厂预装跳舞、踢腿等多种趣味动作程序，支持遥控器控制机器人移动、预设动作表演，还可实现头部仰俯转动，开箱即玩。
