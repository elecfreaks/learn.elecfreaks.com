# 案例一:向前行走

## 开箱指南

1. 从包装材料中取出机器人 PU 和游戏手柄
2. 打开外包装盒，取出产品使用手册
3. 在机器人背包（背部储物仓）中找到配件包，内含 2 节 AAA 电池
4. 找到螺丝刀（用于电池安装）
5. 确认所有物品完好无损

## 安装电池
1. 找到游戏手柄背面的电池盖
2. 使用螺丝刀拧开电池盖螺丝
3. 装入2节AAA电池（注意正负极方向）
4. 盖回电池盖并拧紧螺丝

**安全提醒：**
- 请使用全新碱性电池
- 勿混用新旧电池、不同类型电池
- 长时间不使用请取出电池

## 遥控手柄开机操作
1. 找到手柄侧面的电源开关
2. 将开关拨至 "ON" 档位
3. micro:bit LED 屏显示当前无线电控制频道号

## 关机操作
1. 将电源开关拨至 "OFF" 档位
2. LED 显示屏熄灭

**省电建议：**长时间不使用时，请关闭手柄电源以节省电量

## PU Robot开机操作
1. 找到机器人下巴处的黑色电源键
2. 轻按电源键（短按一下）
3. 等待2秒校准完成
**位置示意：**电源键位于机器人正面下巴位置
