# Firmware Flashing

If you need to update the firmware of your PU Robot, please follow these steps:

1. Get the Firmware

PU Robot Firmware Download:
Remote Controller Firmware Download:

2. Flash the firmware separately to the two micro:bit boards corresponding to the PU Robot and the remote controller.

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

**Note:** Distinguish between the two micro:bit boards to prevent the PU Robot from failing to start due to incorrect firmware.

3. Removal and Installation of micro:bit

    1. Rotate the body of the PU Robot by 90°.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-program-download-01.png)

    2. Unscrew the fixing screws.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-program-download-02.png)

    3. Open the head cover and install the micro:bit.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-program-download-03.png)

    4. Close the head cover and tighten the fixing screws back into place.

**Warning:** Do not forcibly twist the joints of the PU Robot while it is powered on, as this will damage the servo motors.

4. Install the micro:bit into the remote controller

    1. Unscrew the acrylic fixing screws on the back of the controller.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-flashing-firmware-01.png)

    2. Install the micro:bit.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-kit-flashing-firmware-02.png)

    3. Tighten the fixing screws back into place.
