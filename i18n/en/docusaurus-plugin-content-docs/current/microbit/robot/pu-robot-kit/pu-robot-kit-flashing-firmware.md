# Firmware Flashing

If you need to update the firmware of your PU Robot, please follow these steps:

1. Obtain the Firmware

PU Robot Firmware Download:
Remote Controller Firmware Download:

2. Flash the respective firmware to the two micro:bit boards for the PU Robot and the remote controller.

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

**Note:** Distinguish between the two micro:bit boards to avoid incorrect firmware flashing, which may cause the PU Robot to fail to start.

3. Removing and Installing the micro:bit

    1. Rotate the head of the PU Robot by 90°.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    2. Unscrew the fixing screws.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    3. Figure 3: Open the head cover.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    4. Figure 4: Install the micro:bit.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    5. Figure 5: Replace the head cover.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    6. Figure 6: Tighten the fixing screws.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

**Warning:** Do not forcibly twist the joints while the PU Robot is powered on, as this may damage the servo motors.

4. Install the micro:bit into the remote controller

    1. Figure 1: Unscrew the acrylic fixing screws on the back of the controller.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    2. Figure 2: Install the micro:bit.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

    3. Figure 3: Tighten the fixing screws.

    ![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)
