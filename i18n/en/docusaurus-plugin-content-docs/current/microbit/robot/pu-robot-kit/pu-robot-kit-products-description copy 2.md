# 产品简介

## 简介
---

PU 机器人是专为 STEAM 教育打造的智能教育机器人，采用模块化结构设计，兼容积木搭建。支持 GPIO、IIC 等标准接口与传感器扩展，配备 LED 大灯、超声波传感器。出厂预装跳舞、踢腿等多种趣味动作程序，支持遥控器控制机器人移动、预设动作表演，还可实现头部仰俯转动，开箱即玩。

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

## 产品特性

**PU外壳设计**：采用 PU 全包式外壳设计，机身走线规整有序，仅少量线材外露，整体造型简洁一体，防护性与美观度兼具。

**6-DOF**：搭载 6-DOF 运动架构，关节活动灵活自然，可实现舒展流畅的仿生动作，完成更多样化的姿态表现。

**平衡算法**：内置专业平衡算法，行走时通过头部舵机实时转动调整重心偏移，大幅提升行走稳定性，姿态更稳健。

**快速上手**：套装标配专属遥控手柄，机身内置预设程序，开箱通电即可直接操控，无需复杂调试，零基础也能快速体验。

**趣味功能**：集成跳舞、踢腿、避障行走等趣味玩法，支持节奏跟随、随机状态互动，互动体验更丰富。

**易于编程**：同时支持 Python 代码编程与 MakeCode 图形化编程，图形化拖拽操作轻松入门，兼顾新手入门与进阶开发。

**丰富的学习资源**：配套完整且丰富的官方教程，从基础操控到进阶编程开发，全方位助力快速掌握机器人使用与学习技巧。
