# Product Introduction

## Overview
---

The PU Robot is an intelligent educational robot designed specifically for STEAM education. It features a modular structure and is compatible with building block construction. Supporting standard interfaces such as GPIO and IIC for sensor expansion, it is equipped with LED headlights and an ultrasonic sensor.
The robot comes pre-installed with a variety of interesting action programs including dancing and kicking. It supports remote control for movement and preset action performances, and can also perform head tilting rotation. It is ready to play right out of the box.

![](https://wiki-media-ef.oss-cn-hongkong.aliyuncs.com/docs/microbit/robot/pu-robot-kit/pu-robot-01.png)

## Product Features

**PU Shell Design**: Adopts a fully wrapped PU shell with neat internal wiring and only a small number of exposed wires. The overall design is simple and integrated, combining both protection and aesthetic appeal.

**6-DOF**: Equipped with a 6-DOF motion system, the joints move flexibly and naturally, enabling smooth and fluid bionic movements and more diverse posture expressions.

**Balance Algorithm**: Built-in professional balance algorithm. The robot adjusts its center of gravity in real time through the head servo during walking, greatly improving walking stability and delivering a more steady posture.

**Easy to Use**: The kit comes with a dedicated remote controller, and the robot has built-in preset programs. It can be controlled directly after power-on without complicated debugging, making it accessible for beginners.

**Fun Functions**: Integrates interesting functions such as dancing, kicking, and obstacle-avoiding walking. It supports rhythm following and random interactive states for a richer interactive experience.

**Easy Programming**: Supports both Python text programming and MakeCode block-based programming. The drag-and-drop graphical interface is beginner-friendly, suitable for both entry-level and advanced development.

**Abundant Learning Resources**: Complete and rich official tutorials are provided, covering basic operation to advanced programming development, helping users fully master robot operation and learning skills.
